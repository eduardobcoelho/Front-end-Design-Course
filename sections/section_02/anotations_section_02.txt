What is css?:
	It is a language of style's sheets. It's used to define the visual apresentation of documents written in a markup language like HTML or XML. With him, you can atribute colours, fonts famylis, sizes, and many others formatations to HTML tags.
	CSS3 is the most updated version of the CSS.

Sintaxe CSS:
	CSS are write of this way: - h1 { color: #FF9900; } where h1 is the selector, color is the propriety and #FF9900 is the value.

Sintaxe of classes in CSS:
	.class-name { propriety: value; }.

Video lessons 11.
Working with colours:
	The colours in a eletronic dispositive screen that exist colours, they are formed by the combination of light's points RGB(Red, Green, Blue).
	The total absence of light's points results in the black color.
	The fully of light's points - (255, 255, 255) - results in the white color.

Differents ways to write colour code:
	RGB format --> .txt-green { color: rgb(51, 204, 102); } --> 24 bit color (16.777.216 cores)
	Hexadecimal format --> .txt-green { color: #64DD89; } --> 24 bit color (16.777.216 cores)
	Abbreviated hexadecimal format --> .txt-green { color: ; #3C6; } --> #33CC66 --> 12 bit color (4.096)
	Color names --> .txt-laranja { color: orange; } -->  x11 color names
